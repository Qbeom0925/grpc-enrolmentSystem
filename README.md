
## Project Explanation 
*  본 프로젝트는 원격 프로시저 호출 기술 gRPC를 기반으로 학사 시스템을 구축하였다.
<br>

## Project Goal 
*  gRPC 기술에 대한 이해도를 높이고 프로세스간 통신에 대해서 배운다.
<br>
